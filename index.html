Project

Completed Project

TACACS AAA implementation

I implemented TACACS (Terminal Access Controller Access-Control System) AAA (Authentication, Authorization, and Accounting) to enhance network security by centralizing user access control. This setup allows for precise authentication of users, authorization of commands they can execute, and detailed accounting of their actions, providing better auditing and security management. By using TACACS, I can enforce granular control over network device access, ensuring that only authorized personnel can modify device configurations and track their activities.

Completed Project

Radius Server AAA implementation

I set up a RADIUS (Remote Authentication Dial-In User Service) server for AAA (Authentication, Authorization, and Accounting) to manage user access to network resources. The RADIUS server was configured on a Windows Server, where I integrated user credentials and access policies to control and authenticate network device access. This setup allows for centralized authentication and authorization, ensuring only authorized users can access network services while providing detailed accounting for audit and compliance purposes.

Completed Project

LibreNMS Network Monitoring System

I implemented LibreNMS as a network monitoring system to provide real-time visibility into network devices, performance metrics, and health status. With its auto-discovery feature, LibreNMS seamlessly detects and maps devices, allowing for easy monitoring and management of network infrastructure. The system offers customizable alerting, detailed reporting, and extensive support for various network protocols, ensuring proactive monitoring and timely response to any potential issues.
